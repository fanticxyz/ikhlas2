# 🕌 Islamic Knowledge Discord Bot

Full hadith collections (36,000+ hadiths) + entire Quran. No API key required.

## Collections
- 📗 Sahih al-Bukhari — 7,563 hadiths
- 📘 Sahih Muslim — 7,563 hadiths  
- 📙 Sunan Abu Dawud — 5,274 hadiths
- 📕 Jami' al-Tirmidhi — 3,956 hadiths
- 📒 Sunan Ibn Majah — 4,341 hadiths
- 📓 Sunan al-Nasai — 5,758 hadiths
- 📔 Muwatta Imam Malik — 1,832 hadiths
- 🌿 40 Hadith al-Nawawi — 42 hadiths
- ✨ 40 Hadith Qudsi — 40 hadiths
- 📖 Quran — 6,236 ayahs (114 surahs)

## Commands
- `/hadith collection: number:` — Any hadith by number
- `/random` — Random hadith from any/specific collection
- `/browse collection: start:` — Navigate with Prev/Next/Random buttons
- `/ayah surah: ayah:` — Any Quran verse
- `/randomayah` — Random Quran verse
- `/daily` — Daily hadith + ayah (auto-rotates each day)
- `/collections` — List all collections
- `/explore` — Interactive dropdown explorer

## Setup
1. Create bot at discord.com/developers/applications
2. Copy your bot token
3. Run: `npm install`
4. Create `.env` file: `DISCORD_TOKEN=your_token_here`
5. Run: `npm start`

## Sources
- Hadith: github.com/fawazahmed0/hadith-api (free, no key)
- Quran: api.alquran.cloud (free, no key)
